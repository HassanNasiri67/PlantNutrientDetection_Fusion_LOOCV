# Plant Nutrient Deficiency Detection (Fusion + LOOCV)

This repository contains the full Python code and documentation used in the paper:
"Smart Detection of Plant Nutrient Deficiencies Using Machine Learning and Image Fusion."

## Contents
- `Code/`: Python scripts for feature extraction, classification, and fusion.
- `Results/`: Output metrics (Accuracy, Precision, Recall, F1).
- `Dataset/`: Image samples (if permitted by data-sharing policy).
- `Supplementary/`: Figures and highlight image.
